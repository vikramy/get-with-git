Step 3
Make Changes in the Working Directory

  1. Open the file public/app/practice/final.html

    Add your information (name, github user name, and email - ignore favorite sandwich for now) into the empty table row

  2. Create a new file in /public/app/practice/ with the name step-42.html

Switch to your shell and execute the following:

git status

You'll see that final.html is listed as Changes not staged to commit - modified and step-42.html is listed as untracked.

  1. git add public/app/practice/final.html
  2. git status

final.html is now listed as Changes to be committed - modified and step-42.html is still untracked.

  1. git add .
  2. git status

git add . adds all unstaged and unmodified files. Now both files are listed as Changes to be committed... for now.
